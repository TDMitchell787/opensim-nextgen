# 🌟 Quick Start Standalone - Your First Virtual World

Welcome to OpenSim! This template gets you up and running with your own virtual world in just 2 minutes.

## What You Get

🏝️ **Your Own Island**: A beautiful 256x256 meter region called "MyFirstRegion"  
👤 **Admin Powers**: You're automatically the region owner with full control  
🎨 **Creative Freedom**: Build, script, and explore to your heart's content  
💾 **Simple Storage**: Uses SQLite - no complex database setup required  
🔧 **Safe Settings**: Beginner-friendly configuration that just works  

## Quick Start Guide

### Step 1: Deploy This Template
1. Click "Deploy This Setup" in the FWDFE interface
2. Choose a name for your world (or keep the default)
3. Wait for automatic configuration (about 30 seconds)

### Step 2: Start Your World
1. Run the generated startup script: `./start_server.sh`
2. Wait for "Region MyFirstRegion is now ready" message
3. Your world is live on `localhost:9000`!

### Step 3: Connect and Explore
1. Open your Second Life viewer (Firestorm recommended)
2. Add a new grid:
   - Grid Name: "My First World"
   - Login URI: `http://127.0.0.1:8080`
3. Create your avatar account through the web interface
4. Log in and start exploring!

## What Can You Do?

### 🏗️ Building
- Right-click on ground → "Create" to start building
- Use the Build floater to create and modify objects
- Try the built-in shapes: cube, sphere, cylinder, prism

### 📜 Scripting
- Add scripts to objects for interactivity
- LSL (Linden Scripting Language) is enabled
- OSSL functions available for advanced features
- Start with simple scripts like hover text or rotation

### 🌍 Exploring
- Fly around your region (Page Up to fly)
- Change the environment (World → Environment Editor)
- Teleport to different spots within your region

### 👥 Sharing
- Invite friends by giving them your grid address
- They can create accounts and visit your world
- Build together and share creations
